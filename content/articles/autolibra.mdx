# Inspired by human self-regulated learning

In the learning interaction, children understand the adult regulation (instruction),
but they do so in relation to their own task understanding, which requires a coordinating of the two
perspectives. The cognitive representation that results,
therefore, is not just of the instructions but of the inter-subjective dialogue.

When human kids learn from instructions from adults, they coordinate two views
from others during social or cultural learning <Citation id="tomasello1993cultural"/>.
These instructions and feedback are internalized for self-regulation
<Citation id="pintrich2002development"/><Citation id="nicol2006formative"/>,
providing intrinsic rewards for continuous improvement using new experiences.
That is, the feedback for the past experience, is generalized and rewards future experience.
Drawing inspiration from this process, we introduce AutoLibra, which builds an evaluation system
<EmphasizedSpan>from open-ended human feedback through induction of generalizable metrics</EmphasizedSpan>.


<Callout icon="üî¨">
    AutoLibra can be used as a lens to provide grounded, behavior-salient insights into agent trajectories.
    We find that it induces metrics which are <EmphasizedSpan> concrete than the metrics proposed by experts</EmphasizedSpan> and
    the ones which <EmphasizedSpan>have been overlooked by experts</EmphasizedSpan>.
</Callout>

<Callout icon="ü™ú">
    AutoLibra-derived metrics are also <EmphasizedSpan>great optimization targets</EmphasizedSpan> for both human prompt engineers and automatic agent finetuning.
</Callout>
